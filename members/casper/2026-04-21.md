### 📝 Tóm tắt 5 câu đã làm
1. **Câu 1:** A company wants to run its critical applications in containers to meet requirements for scalability and availability. The company prefers to focus on maintenance of the critical applications. The company does not want to be responsible for provisioning and managing the underlying infrastructure that runs the containerized workload.
What should a solutions architect do to meet these requirements?
- Kiến thức rút ra:
    Use Amazon Elastic Container Service (Amazon ECS) on AWS Fargate.
    ECS cho chạy container và fargate để k quan tâm infa.

2. **Câu 2:** A company has a data ingestion workflow that consists of the following:
• An Amazon Simple Notification Service (Amazon SNS) topic for notifications about new data deliveries
• An AWS Lambda function to process the data and record metadata
The company observes that the ingestion workflow fails occasionally because of network connectivity issues. When such a failure occurs, the Lambda function does not ingest the corresponding data unless the company manually reruns the job.
Which combination of actions should a solutions architect take to ensure that the Lambda function ingests all data in the future? (Choose two.)
- Kiến thức rút ra:
    Create an Amazon Simple Queue Service (Amazon SQS) queue, and subscribe it to the SNS topic.
    Modify the Lambda function to read from an Amazon Simple Queue Service (Amazon SQS) queue.
    Tạo SQS sub SNS để nhận message .
    Lambda sẽ đọc SQS để xlí , có DLD để retry và visible tránh trùng.

3. **Câu 3:** An application development team is designing a microservice that will convert large images to smaller, compressed images. When a user uploads an image through the web interface, the microservice should store the image in an Amazon S3 bucket, process and compress the image with an AWS Lambda function, and store the image in its compressed form in a different S3 bucket.
A solutions architect needs to design a solution that uses durable, stateless components to process the images automatically.
Which combination of actions will meet these requirements? (Choose two.)
- Kiến thức rút ra:
    Create an Amazon Simple Queue Service (Amazon SQS) queue. Configure the S3 bucket to send a notification to the SQS queue when an image is uploaded to the S3 bucket.
    Configure the Lambda function to use the Amazon Simple Queue Service (Amazon SQS) queue as the invocation source. When the SQS message is successfully processed, delete the message in the queue.


4. **Câu 4:** A company is migrating a distributed application to AWS. The application serves variable workloads. The legacy platform consists of a primary server that coordinates jobs across multiple compute nodes. The company wants to modernize the application with a solution that maximizes resiliency and scalability.
How should a solutions architect design the architecture to meet these requirements?
- Kiến thức rút ra:
    Configure an Amazon Simple Queue Service (Amazon SQS) queue as a destination for the jobs. Implement the compute nodes with Amazon EC2 instances that are managed in an Auto Scaling group. Configure EC2 Auto Scaling based on the size of the queue.
    Dùng SQS để nhận job , tách biệt hệ thống, các server khác polling từ SQS và tư động scale vs ASG.
    Tuy nhiên size sẽ đc phụ thuộc vào queue,
    Cloudwatch có thể monitor số lượng message, nếu tồn queue nhiều thì scale in để xử lí.
    K thể dùng schedule scaling vì lượng workload là biến dổi liên tục.

5. **Câu 5:** A company runs multiple Windows workloads on AWS. The company's employees use Windows file shares that are hosted on two Amazon EC2 instances. The file shares synchronize data between themselves and maintain duplicate copies. The company wants a highly available and durable storage solution that preserves how users currently access the files.
What should a solutions architect do to meet these requirements?
- Kiến thức rút ra:
    Extend the file share environment to Amazon FSx for Windows File Server with a Multi-AZ configuration. Migrate all the data to FSx for Windows File Server.
    Để chia sẻ file, migrate , sync nhanh thì dùng FSx for windows vì tương thích hoàn toàn.
    EFS có thể sync tương tự nhưng chỉ hỗ trợ linux.
