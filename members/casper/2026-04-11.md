### 📝 Tóm tắt 5 câu đã làm
1. **Câu 1:** A company is deploying a new public web application to AWS. The application will run behind an Application Load Balancer (ALB). The application needs to be encrypted at the edge with an SSL/TLS certificate that is issued by an external certificate authority (CA). The certificate must be rotated each year before the certificate expires.
What should a solutions architect do to meet these requirements?

- Kiến thức rút ra:
    Cần phải có khả năng renew cert.
    Tuy nhiên, external là bên thứ 3, k phải do AWS tạo nên k thể auto renew được.
    Do đó phải import vào ACM, để có data và trigger bằng eventbridge để biết khi nào hết hạn để renew thủ công.

2. **Câu 2:** A company has an on-premises application that generates a large amount of time-sensitive data that is backed up to Amazon S3. The application has grown and there are user complaints about internet bandwidth limitations. A solutions architect needs to design a long-term solution that allows for both timely backups to Amazon S3 and with minimal impact on internet connectivity for internal users.
Which solution meets these requirements?
- Kiến thức rút ra:
    Do time cần gấp và data liên tục vừa backup vừa chạy app.
    Dẫn đến việc chậm bandwidth từ internet -> S3.
    giải pháp xử lí sẽ là cần 1 direct connect riêng từ on-prem -> S3 vì nó là đường riêng nên k chung vs public internet.
    Vẫn có thể backup nhưng hạn chế limit do tách riêng.

3. **Câu 3:** A development team needs to host a website that will be accessed by other teams. The website contents consist of HTML, CSS, client-side JavaScript, and images.
Which method is the MOST cost-effective for hosting the website?
- Kiến thức rút ra:
    Vì cần web dạng static kiểu thuần html, css, js, images , ...
    Vì vậy chỉ cần đặt data vào S3 rồi host S3 lên static web.

4. **Câu 4:** A company needs to store its accounting records in Amazon S3. The records must be immediately accessible for 1 year and then must be archived for an additional 9 years. No one at the company, including administrative users and root users, can be able to delete the records during the entire 10-year period. The records must be stored with maximum resiliency.
Which solution will meet these requirements?
- Kiến thức rút ra:
    Cần lưu trữ lâu dài mà k cần xem thì dùng glacier deep archive để tiết kiệm.
    Cần lifecycle policy để auto move sau 1 năm, và thêm object lock để chặn xoá.

5. **Câu 5:** A company collects data for temperature, humidity, and atmospheric pressure in cities across multiple continents. The average volume of data that the company collects from each site daily is 500 GB. Each site has a high-speed Internet connection.
The company wants to aggregate the data from all these global sites as quickly as possible in a single Amazon S3 bucket. The solution must minimize operational complexity.
Which solution meets these requirements?
- Kiến thức rút ra:
    Vì cần tốc độ cao và đơn giản , ít effort.
    Thì càng thay đổi ít càng tốt.
    Chỉ cần dùng S3 transfer acceleration ở S3 bucket để có thể upload nhanh tới S3 vì dùng edge connection cho gần, network tốt, latency thấp.
    Và bật multipart upload để up song song file nặng tầm 500GB.
