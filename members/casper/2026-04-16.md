### 📝 Tóm tắt 5 câu đã làm
1. **Câu 1:** A company has an Amazon S3 bucket that contains critical data. The company must protect the data from accidental deletion.
Which combination of steps should a solutions architect take to meet these requirements? (Choose two.)

- Kiến thức rút ra:
    Bật cả versioning để lưu nhiều version, nếu xoá nhầm cũng sẽ là deletion maker, có thể restore.
    Bật MFA để thêm lớp xử lí khi muốn xoá.

2. **Câu 2:** An application runs on an Amazon EC2 instance in a VPC. The application processes logs that are stored in an Amazon S3 bucket. The EC2 instance needs to access the S3 bucket without connectivity to the internet.
Which solution will provide private network connectivity to Amazon S3?
- Kiến thức rút ra:
    Tạo gateway VPC endpoint to the S3 bucket.
    Gateway VPC endpoint sẽ tự tạo route table để mapping EC2 tới S3 đường private.

3. **Câu 3:** A company's website uses an Amazon EC2 instance store for its catalog of items. The company wants to make sure that the catalog is highly available and that the catalog is stored in a durable location.
What should a solutions architect do to meet these requirements?
- Kiến thức rút ra:
    Move data catalog vào EFS system.
    EFS tách biệt vs EC2, multi-AZ nên tăng HA và durable.
    EFS hỗ trợ đọc.ghi từ nhiều instance.

4. **Câu 4:** A company wants to improve its ability to clone large amounts of production data into a test environment in the same AWS Region. The data is stored in Amazon EC2 instances on Amazon Elastic Block Store (Amazon EBS) volumes. Modifications to the cloned data must not affect the production environment. The software that accesses this data requires consistently high I/O performance.
A solutions architect needs to minimize the time that is required to clone the production data into the test environment.
Which solution will meet these requirements?
- Kiến thức rút ra:
    Để clone data từ prod -> test , k impact thì cần tách riêng data EBS riêng.
    -> Tạo EBS snapshot -> dùng chức năng esb fast snapshot restore để restore nhanh -> sau đó attach vào EC2 dev.

5. **Câu 5:** A company wants to migrate its on-premises application to AWS. The application produces output files that vary in size from tens of gigabytes to hundreds of terabytes. The application data must be stored in a standard file system structure. The company wants a solution that scales automatically. is highly available, and requires minimum operational overhead.
Which solution will meet these requirements?
- Kiến thức rút ra:
    Do cần lưu file chuẩn file system thì cần EFS chứ k thể S3.
    EBS cũng k đc do k multi attach được.
    Vậy cần migrate app -> EC2 và bật multi AZ để HA, sau đó sync file về EFS.
