### 📝 Tóm tắt 5 câu đã làm
1. **Câu 1:** A company is hosting a static website on Amazon S3 and is using Amazon Route 53 for DNS. The website is experiencing increased demand from around the world. The company must decrease latency for users who access the website.
Which solution meets these requirements MOST cost-effectively?
- Kiến thức rút ra:
    Muốn phân phối static lên over world và giảm latency thì phải cần CDN.
    Host SE -> Cloudfront để CDN -> Route 53 point về CDN.

2. **Câu 2:** A company is building an application in the AWS Cloud. The application will store data in Amazon S3 buckets in two AWS Regions. The company must use an AWS Key Management Service (AWS KMS) customer managed key to encrypt all data that is stored in the S3 buckets. The data in both S3 buckets must be encrypted and decrypted with the same KMS key. The data and the key must be stored in each of the two Regions.
Which solution will meet these requirements with the LEAST operational overhead?
- Kiến thức rút ra:
    Create a customer managed multi-Region KMS key. Create an S3 bucket in each Region. Configure replication between the S3 buckets. Configure the application to use the KMS key with client-side encryption.
    Để multi region dùng cùng 1 KMS key thì KMS cũng phải sử dụng multi-region.
    S3 bật cross region để auto sync data.
    App dùng KMS để mã hoá data trước khi upload.

3. **Câu 3:** A company stores call transcript files on a monthly basis. Users access the files randomly within 1 year of the call, but users access the files infrequently after 1 year. The company wants to optimize its solution by giving users the ability to query and retrieve files that are less than 1-year-old as quickly as possible. A delay in retrieving older files is acceptable.
Which solution will meet these requirements MOST cost-effectively?
- Kiến thức rút ra:
    Sử dụng Inteligent-tiering để tự đọng di chuyển file giữa các plan (frequence vs infrequence).
    Dùng lifecycle để move file to glacier sau 1 năm.
    Query S3 để lấy dtâ vs Athena.
    Query S3 Galicer lấy data vs Galacier select.

4. **Câu 4:** A company provides a Voice over Internet Protocol (VoIP) service that uses UDP connections. The service consists of Amazon EC2 instances that run in an Auto Scaling group. The company has deployments across multiple AWS Regions.
The company needs to route users to the Region with the lowest latency. The company also needs automated failover between Regions.
Which solution will meet these requirements?
- Kiến thức rút ra:
    App VoIP dùng UDP thì cần NLB (layer 4).
    Vậy sẽ phải tạo NLB associate với ASG,.
    Sau đó dùng NLB vs mode Global accelerator để endpoint nhanh, latency thấp tới region do Global Accelerator hỗ trợ any cast IP, support NLB làm endpoint.

5. **Câu 5:** A company runs a highly available image-processing application on Amazon EC2 instances in a single VPC. The EC2 instances run inside several subnets across multiple Availability Zones. The EC2 instances do not communicate with each other. However, the EC2 instances download images from Amazon S3 and upload images to Amazon S3 through a single NAT gateway. The company is concerned about data transfer charges.
What is the MOST cost-effective way for the company to avoid Regional data transfer charges?
- Kiến thức rút ra:
    Tốn kém là do EC2 connect S3 qua NAT thì cần qua internet tốn data out.
    Vậy tạo gateway VPC endpoint để EC2 chọc thẳng vào S3 theo private subnet, tiét kiêm data out internet.
