# Amazon RDS (Relational Database Service):
> Định nghĩa: Cơ sở dữ liệu quan hệ có quản lý (MySQL, PostgreSQL, SQL Server...).

> Best Practice: Dùng Multi-AZ cho môi trường Production (để dự phòng thảm họa). Dùng Read Replicas để giảm tải truy vấn đọc (SELECT) cho database chính.

# AWS WAF (Web Application Firewall): 
> Định nghĩa: Tường lửa lớp 7, bảo vệ ứng dụng web khỏi các tấn công như SQL Injection, Cross-Site Scripting (XSS).

# AWS Shield: 
> Định nghĩa: Dịch vụ chống DDoS (Distributed Denial of Service). Bản Standard được miễn phí và tự động bật cho mọi khách hàng AWS.

# AWS KMS (Key Management Service): 
> ĐỊnh nghĩa: Dịch vụ quản lý khóa mã hóa.

> Best Practice: Mã hóa dữ liệu mọi lúc: "At rest" (khi lưu trữ) và "In transit" (khi truyền tải).
